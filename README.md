
# Portfólio Pessoal

## Descrição
Este projeto foi desenvolvido como atividade da disciplina de Introdução a Computação. O objetivo é se apresentar como um estudante e profissional da área, mas demonstrando sua personalidade e habilidades de forma transparente.

## Funcionalidades

* Navegação entre páginas utilizando links.
* uso de diferentes tags no html e no css
* Layout responsivo desenvolvido com Flexbox.
* Recurso em JavaScript utilizado para mostrar o horário em tempo real.

## Tecnologias Utilizadas

* HTML
* CSS3
* JavaScript

## Estrutura do Projeto

* index.html – Página inicial.
* projetos.html – Página de projetos e certificados.
* sobre.html - Página descrevendo sobre mim
* css/style.css – Estilos do site.
* js/script.js – Função do javaS para horário.
